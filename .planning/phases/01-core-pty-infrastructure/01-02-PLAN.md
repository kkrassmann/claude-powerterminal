Create killPtyProcess function with graceful-then-force pattern:

Function signature:
```typescript
export async function killPtyProcess(ptyProcess: IPty, timeoutMs = 3000): Promise<void>
```

Implementation steps:
1. Get PID from ptyProcess.pid
2. Call ptyProcess.kill() for graceful termination attempt
3. Wait for process exit with timeout:
   - Create exitPromise that resolves on ptyProcess.onExit()
   - Create timeoutPromise that resolves after timeoutMs
   - Use Promise.race([exitPromise, timeoutPromise])
4. If timeout expires and process still running, force-kill on Windows:
   - Check `process.platform === 'win32'`
   - Execute `taskkill /PID ${pid} /T /F` using child_process.exec
   - /T flag terminates process tree (kills conhost.exe children)
   - /F flag forces termination
   - Wrap in try/catch (taskkill errors if process already gone)
5. Log outcomes (graceful success, force-kill success, errors)

Import requirements:
- `import { IPty } from 'node-pty'`
- `import { exec } from 'child_process'`
- `import { promisify } from 'util'`
- `const execAsync = promisify(exec)`

Error handling:
- Don't throw on taskkill errors (process may already be terminated)
- Log warnings for unexpected errors
- Check error message for "not found" or "no tasks" (expected when process already dead)
