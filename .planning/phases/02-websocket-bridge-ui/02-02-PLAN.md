4. **Create `src/src/app/components/terminal/terminal.component.ts`:**

   This is the main component. It must be a standalone Angular component.

   **Imports:**
   - Angular: `Component`, `Input`, `OnInit`, `OnDestroy`, `ViewChild`, `ElementRef`, `NgZone`
   - xterm: `Terminal` from `@xterm/xterm`, `FitAddon` from `@xterm/addon-fit`, `WebglAddon` from `@xterm/addon-webgl`, `WebLinksAddon` from `@xterm/addon-web-links`
   - Protocol: `ServerMessage`, `ClientMessage`, `WS_PORT` from `../../../../shared/ws-protocol`

   **Component metadata:**
   ```typescript
   @Component({
     selector: 'app-terminal',
     standalone: true,
     template: `<div #terminalContainer class="terminal-container"></div>`,
     styleUrls: ['./terminal.component.css']
   ```

   **Properties:**
   - `@Input() sessionId!: string` — session to connect to
   - `@ViewChild('terminalContainer', { static: true }) terminalContainer!: ElementRef<HTMLDivElement>`
   - `private term!: Terminal`

   - `private socket!: WebSocket`

   - Load WebLinksAddon: `this.term.loadAddon(new WebLinksAddon());`
   - Open terminal: `this.term.open(this.terminalContainer.nativeElement);`
   - Load WebGL addon with DOM fallback (MUST be after `open()`):
     ```typescript
     try {
       const webglAddon = new WebglAddon();
       webglAddon.onContextLoss(() => {
         webglAddon.dispose();
         // DOM renderer activates automatically as fallback
       });
       this.term.loadAddon(webglAddon);
     } catch (e) {
       console.warn('[Terminal] WebGL unavailable, using DOM renderer');
     }
     ```
   - Fit after open: `this.fitAddon.fit();`
   - Setup clipboard handling (see below)

   **setupClipboard():**
   Per CONTEXT.md: Ctrl+C copies when selected, SIGINT when not. Ctrl+V handled by browser. All other keys pass through.
   ```typescript
   this.term.attachCustomKeyEventHandler((event: KeyboardEvent) => {
     if (event.type !== 'keydown') return true;

     if (event.ctrlKey && event.code === 'KeyC') {
       if (this.term.hasSelection()) {
         navigator.clipboard.writeText(this.term.getSelection());
         this.term.clearSelection();
         return false; // Prevent xterm from sending \x03
       }
       return true; // No selection: send SIGINT
     }

     if (event.ctrlKey && event.code === 'KeyV') {
       return false; // Let browser handle paste
     }

     return true; // All other keys pass through to PTY
   });
   ```

   **connectWebSocket():**
   - Create WebSocket: `this.socket = new WebSocket(\`ws://localhost:${WS_PORT}/terminal/${this.sessionId}\`)`
   - `onopen`: Reset `reconnectAttempts = 0`. Log connected.
   - `onmessage`: Parse JSON as ServerMessage. Switch on type:
     - `'buffering'`: Set `this.isBuffering = true`. Call `this.term.reset()` to clear terminal before replay (prevents Pitfall 3: duplicate content).
     - `'buffered'`: Set `this.isBuffering = false`. Log replay complete.
     - `'output'`: Call `this.term.write(msg.data)`.
     - `'exit'`: Call `this.term.write('\r\n[Process exited with code ' + msg.exitCode + ']\r\n')`. Do NOT reconnect.
   - `onclose`: If not `this.destroyed` and not after an exit message, call `this.scheduleReconnect()`.
   - `onerror`: Log error. `onclose` fires after `onerror`.
   - Forward user input to WebSocket:
     ```typescript
     this.term.onData((data: string) => {
       if (this.socket?.readyState === WebSocket.OPEN) {
         const msg: ClientMessage = { type: 'input', data };
         this.socket.send(JSON.stringify(msg));
       }
     });
     ```

   **scheduleReconnect():**
   Exponential backoff with jitter (per CONTEXT.md: silent auto-reconnect):
   ```typescript
   const baseDelay = 1000;
   const delay = Math.min(baseDelay * Math.pow(2, this.reconnectAttempts), this.maxReconnectDelay);
   const jitter = delay * 0.1 * (Math.random() - 0.5);
   setTimeout(() => {
     this.reconnectAttempts++;
     this.connectWebSocket();
   }, delay + jitter);
   ```

   **setupResizeHandler():**
   ResizeObserver on container, debounced at 200ms (per CONTEXT.md locked decision):
   ```typescript
   this.resizeObserver = new ResizeObserver(() => {
     clearTimeout(this.resizeTimeout);
     this.resizeTimeout = setTimeout(() => {
       this.fitAddon.fit();
       if (this.socket?.readyState === WebSocket.OPEN) {
         const msg: ClientMessage = { type: 'resize', cols: this.term.cols, rows: this.term.rows };
         this.socket.send(JSON.stringify(msg));
       }
     }, 200);
   });
   this.resizeObserver.observe(this.terminalContainer.nativeElement);
   ```

   **ngOnDestroy():**
   Clean up everything to prevent memory leaks (Pitfall 7):
   - `this.destroyed = true`
   - `clearTimeout(this.resizeTimeout)`
   - `this.resizeObserver?.disconnect()`
   - `this.socket?.close()`
   - `this.term?.dispose()`
